Drawing multiple points:

  1. In previous example we add one point to the canvas. Now let's add multiple points.
  2. We added an attribute with name a_Position to the vertex shader section, to draw multiple points.
  3. initVertexBuffers function creates vertex buffer, binds the buffer object to the target and returns the number of vertices/points.
  4. Vertices are written into buffer using gl.bufferData(must read).
  5. To render three poitns on to the canvas, we use gl.drawArrays(,..) method, which takes the type of primitive to render as the first argument, the starting index in the array of vector points as the second argument and number of indices to be rendered as the third argument.

    Ex: // Draw three points
        gl.drawArrays(gl.POINTS, 0, 3);

Please use a browser that supports "canvas"
